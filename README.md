Circular Buffer Implementation

هذا المشروع هو عبارة عن تنفيذ لهيكل بيانات المخزن الدائري بلغة C.
يتضمن الكود العمليات الأساسية:
* التهيئة (Initialization)
* الإضافة (Write) مع معالجة حالة الامتلاء (Overflow)
* القراءة (Read) مع معالجة حالة الفراغ (Underflow)

تم إنجاز هذا العمل كجزء من تكليفات CE-ESY26.
